# How to view your Casa de Corona site

Localhost isn’t working on your machine, so use one of these instead.

---

## Option 1: Deploy to Vercel (recommended — get a real link in ~2 minutes)

Your project is already on GitHub at **rudycoronaonline-wq/casa-de-corona**. Deploy it to get a link that works on any device.

1. Go to **https://vercel.com** and sign in with **GitHub**.
2. Click **“Add New…”** → **“Project”**.
3. Under **Import Git Repository**, find **casa-de-corona** and click **Import**.
4. Click **Deploy** (don’t change any settings).
5. When it finishes, click the link Vercel shows (e.g. **https://casa-de-corona-xxx.vercel.app**). That’s your live site.

You can share that link with anyone; it will always open the site.

---

## Option 2: Open the file directly in your browser

1. Press **Windows + E** to open File Explorer.
2. Paste this into the address bar and press Enter:
   ```
   c:\Users\mrrud\OneDrive\Desktop\Website build\Ventanas-Website\casa-de-corona
   ```
3. **Double-click `index.html`**.

It should open in your default browser. If it doesn’t, right‑click `index.html` → **Open with** → **Chrome** (or **Edge** / **Firefox**).

---

## Option 3: GitHub Pages (free public URL)

If you prefer GitHub Pages instead of Vercel:

1. On GitHub, open **rudycoronaonline-wq/casa-de-corona**.
2. Go to **Settings** → **Pages**.
3. Under **Source**, choose **Deploy from a branch**.
4. Branch: **main**, folder: **/ (root)**. Click **Save**.
5. After a minute or two, your site will be at:
   **https://rudycoronaonline-wq.github.io/casa-de-corona/**
